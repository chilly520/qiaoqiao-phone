### 2. 修复经期预测小组件 ✅

#### 问题诊断
- ❌ **原问题**：日历网格显示混乱，日期数字不显示
- ❌ **原因**：
  1. 日期数字显示条件过于严格（只在特殊日期显示）
  2. 字体大小不一致
  3. 颜色分类逻辑混乱

#### 修复方案
1. **显示所有日期数字**
   - 只要 `day.isCurrentMonth === true` 就显示数字
   - 不再限制只在特殊日期显示

2. **优化颜色逻辑**
   - 新增 `getDayTextClass()` 函数
   - 根据日期状态返回对应颜色：
     - 当前经期：白色文字
     - 预测经期：玫瑰色文字
     - 排卵期：紫色文字
     - 普通日期：灰色文字

3. **调整布局**
   - 网格间距从 `gap-1.5` 改为 `gap-1`（更紧凑）
   - 字体大小从 `text-[8px]` 改为 `text-[9px]`（更清晰）
   - 去除背景色的透明度，使用纯色

#### 修复效果
```
┌─────────────────────────┐
│  经期          🌸       │
│                         │
│  ○ ○ ● ● ● ○ ○         │ ← 日历网格
│  ○ ○ ○ ○ ○ ○ ○         │    ● = 特殊日期
│  ○ ○ ○ ○ ○ ○ ○         │    ○ = 普通日期
│                         │
│      距离预测经期 9 天   │
└─────────────────────────┘
```

### PeriodWidget 修复
```javascript
// 修复前：只有特殊日期才显示数字

  {{ day.day }}
```
