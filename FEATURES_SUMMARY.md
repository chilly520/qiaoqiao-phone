## 🆕 v1.3.67 更新内容

### 新增功能
1. ✅ **物流手动签收**
   - 物流派送完成后不自动签收
   - 用户需手动点击"确认签收"按钮
   - 防止误操作，提升用户体验

2. ✅ **签收后商品存入背包**
   - 签收时自动将商品加入微信背包
   - 支持多商品批量存入
   - 背包物品可查看详情

3. ✅ **订单商品点击查看详情**
   - 订单列表中的商品可点击
   - 弹出商品详情弹窗
   - 支持加入购物车/立即购买

4. ✅ **已完成订单查看物流**
   - 已完成订单保留物流信息
   - 随时可查看历史订单物流
   - 物流轨迹完整保留

### 优化功能
1. ✅ **物流地图节点防遮挡**
   - 智能标注定位系统
   - 左侧节点标注在右边
   - 右侧节点标注在左边
   - 中间节点标注在上方居中

2. ✅ **节点大小优化**
   - 节点光晕半径：5 → 3
   - 外圈半径：3 → 2
   - 标注框尺寸缩小
   - 字体大小优化 (3.5/3.2 → 2.8/2.6)

3. ✅ **文字居中优化**
   - 所有标注框文字居中对齐
   - 收货地址文字完美居中
   - 框宽度自适应调整

4. ✅ **底部边距优化**
   - 我的页面底部边距增加
   - 服务说明不再被导航栏遮挡
   - 滚动区域留足安全距离
