# 经期追踪功能增强

## 功能概述

本次更新为日历模块添加了完整的经期追踪功能，包括：
1. 个性化周期设置
2. 经期天数设置
3. 自动高亮显示经期范围

## 新增功能

### 1. 经期周期设置弹窗

**入口位置**：右侧面板 -> 快捷工具 -> "周期设置"按钮

**功能特性**：
- 📏 **周期长度设置**（间隔天数）
  - 范围：20-45 天
  - 默认值：28 天
  - 使用滑块直观调节
  - 实时显示是否在正常范围（21-35 天）

- 💧 **经期天数设置**
  - 范围：2-10 天
  - 默认值：5 天
  - 使用滑块直观调节
  - 实时显示是否在正常范围（3-7 天）

- 📊 **统计数据展示**
  - 已记录周期数量
  - 预测周期数量

- 💡 **温馨提示**
  - 周期长度说明
  - 经期天数说明
  - 自动预测功能说明
  - 数据修正说明

### 2. 记录经期弹窗增强

**入口位置**：右侧面板 -> 快捷工具 -> "记录经期"按钮

**新增功能**：
- ⚙️ **周期设置按钮**
  - 在记录经期时可直接打开周期设置
  - 方便在记录过程中调整参数

### 3. 经期范围高亮显示

**显示规则**：
- 实际记录的经期：使用粉红色渐变背景
- 预测的经期：使用橙色渐变背景
- 根据设置的经期天数自动高亮对应日期

**视觉效果**：
- 经期内：粉红色渐变背景
- 预测期：橙色渐变背景
- 底部彩色标记条（3px）

## 使用方法

### 首次使用
1. 打开日历页面
2. 点击右侧"周期设置"按钮
3. 设置你的周期长度（建议参考历史记录）
4. 设置经期天数（通常 3-7 天）
5. 点击"保存设置"

### 记录经期
1. 点击"记录经期"按钮
2. 选择开始和结束日期
3. 记录症状、流量、心情等信息
4. 保存后系统会自动高亮这些天

### 查看高亮
- 日历视图中，经期内显示粉红色背景
- 预测经期显示橙色背景
- 底部有彩色标记条

## 数据说明

### 周期长度（平均周期）
- 定义：两次经期第一天之间的天数
- 正常范围：21-35 天
- 平均值：28 天
- 超出范围会有提示

### 经期天数（平均经期）
- 定义：每次经期持续的天数
- 正常范围：3-7 天
- 平均值：5 天
- 超出范围会有提示

## 预测机制

系统根据以下数据自动预测未来 6 个月的经期：
1. 最后一次经期开始日期
2. 设置的周期长度
3. 设置的经期天数

预测会在以下情况更新：
- 修改周期设置
- 记录新的经期
- 删除经期记录

## 样式设计

### 颜色方案
- 经期：粉红色渐变 (#ffb7c5 → #ff9eb5)
- 预测：橙色渐变 (#ffd8a8 → #ffe4c4)
- 标记条：粉红色渐变

### 交互设计
- 滑块拖拽调节
- 实时数值显示
- 状态颜色提示（正常/警告）
- 平滑过渡动画

## 注意事项

1. 建议先设置好周期参数再记录经期
2. 可在记录时修正实际数据
3. 系统会自动学习并优化预测
4. 数据保存在本地，不会丢失
