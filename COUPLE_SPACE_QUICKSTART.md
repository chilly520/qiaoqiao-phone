# 💕 情侣空间 - 快速开始指南

## 🚀 立即体验

### 1. 启动应用

开发服务器已在运行：
```
http://localhost:5178/
```

### 2. 访问情侣空间

**方式一：通过主页图标**
1. 打开应用首页
2. 找到"情侣空间"图标（💕）
3. 点击进入

**方式二：直接路由**
```
http://localhost:5178/couple
```

## 📱 使用流程

### 第一次使用

#### Step 1: 选择角色
首次进入会弹出角色选择框：
- **小奶狗**：温柔体贴，粘人可爱
- **霸道总裁**：高冷强势，只宠你一人
- **阳光学长**：温暖可靠，永远守护你

点击选择一个角色，然后点击"💌 发送邀请"

#### Step 2: 发送邀请
选择角色后会自动跳转到微信聊天界面，并显示邀请卡片：
```
💌 情侣空间邀请
我想和你开通专属情侣空间
记录我们的甜蜜时光
[💕 同意开通]
```

#### Step 3: 同意开通
点击"同意开通"按钮，显示契约达成卡片：
```
💗 契约达成 💗
情侣空间开通证书
我 💕 角色名
开通日期：2026-03-12
相恋天数：0 天
空间编号：LS202603120001
💕 永久有效
```

#### Step 4: 进入空间主页
成功开通后，进入情侣空间主页，可以看到：

**顶部羁绊区**：
- 双人头像 + 旋转的爱心连线
- 相恋天数："💕 相恋第 X 天"
- 纪念日倒计时（如果有）

**功能网格**（10 个功能）：
```
💌 甜蜜留言    📔 交换日记    🎉 纪念日
👣 一日足迹    📝 便利贴      ✉️ 写信
🏠 两人小屋    ❓ 灵魂提问    📷 相册
🎰 扭蛋机
```

**魔法生成按钮**：右下角悬浮按钮 ⭐

## 🎮 功能演示

### 交换日记 📔

1. 点击"交换日记"卡片
2. 点击右上角"✨"魔法按钮（AI 生成）
3. 或点击"+ 写一篇日记"手动创建
4. 填写内容：
   - 日期选择
   - 天气图标（☀️ ☁️ 🌧️ 等）
   - 心情表情（😊 😢 😠 等）
   - 日记内容
5. 保存后可以在列表中查看

**特色**：
- AI 可以帮你写日记（点击魔法按钮）
- 你和角色都可以写日记
- 支持双向编辑（可以修改任何日记）

---

### 甜蜜留言 💌

1. 点击"甜蜜留言"卡片
2. 点击右上角"+"按钮
3. 选择便签颜色（8 种马卡龙色）
4. 写下想对 TA 说的话
5. 点击"发送"

**效果**：
- 留言像便利贴一样贴在墙上
- 每张便签颜色随机
- 便签有轻微的旋转角度
- 鼠标悬浮时会放大

## 🎨 UI 设计特点

### 色彩系统
- **主色**：粉色系 (#ff6b9d, #ffb7c5)
- **背景**：渐变粉白 (#fff5f7 → #ffe6eb)
- **文字**：深灰/暖棕 (#5a5a7a, #8b7aa8)
- **禁止**：纯黑色

### 动效细节
- ❤️ 心跳动画：爱心持续跳动
- 🔄 旋转动画：戒指和印章缓慢旋转
- ⬆️ 悬浮效果：卡片悬浮时向上移动
- 🔍 缩放交互：鼠标悬浮时轻微放大

### 卡片风格
- 圆角：16px
- 阴影：柔和的粉色阴影
- 毛玻璃：backdrop-filter 模糊效果

## 💾 数据存储

### 本地存储 (LocalStorage)

所有数据都保存在你的浏览器本地：

```javascript
// 查看存储的数据
console.log(localStorage.getItem('loveSpace'))

// 数据结构
{
  "initialized": true,
  "partner": { "id": 1, "name": "小奶狗", ... },
  "startDate": "2026-03-12T06:03:50.000Z",
  "loveDays": 0,
  "diary": [...],
  "messages": [...],
  // ... 其他数据
}
```

### 清除数据

如果想重新开始：
```javascript
localStorage.removeItem('loveSpace')
location.reload()
```

```javascript
store.saveToLocalStorage()
```
