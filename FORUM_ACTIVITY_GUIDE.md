# 🎮 论坛角色活跃度设置 - 快速使用指南

## 📱 如何设置角色活跃度

### 步骤 1：进入绑定页面
```
论坛 App → 底部"我的" → 绑定专属角色
```

### 步骤 2：勾选角色
- 点击角色列表中的 **复选框** ✓
- 支持多选（建议 2-4 个角色）
- 已选中的角色会显示 **蓝色边框 + 浅蓝背景**

### 步骤 3：设置活跃度
选中角色后，右侧会自动出现活跃度下拉框：
```
🔥 高活跃  → 经常发言，积极搞事
💬 中活跃  → 适度发言，偶尔参与
🌙 低活跃  → 很少发言，关键时刻出现
```

## ⚙️ 工作原理

### 数据流向
```
用户勾选角色 → 保存到 forumStore.boundCharIds
     ↓
设置活跃度 → 保存到 chatStore.chats[角色 id].forumActivity
     ↓
AI 生成帖子时读取活跃度 → 控制发言频率
```

### AI 生成逻辑
```javascript
高活跃 (high)    → "经常发言，积极参与话题"
中活跃 (medium)  → "中等活跃度"
低活跃 (low)     → "低活跃，很少发言"
```

## 🔍 效果验证

### 刷新论坛动态后
- **高活跃角色**：频繁出现在帖子和评论中
- **中活跃角色**：适度参与，不会太突兀
- **低活跃角色**：偶尔出现，存在感较低

## ⚠️ 注意事项

### ✅ 正确做法
1. **真人马甲**：务必在"我的马甲衣橱"标记为 `isRealUser`
2. **角色数量**：建议绑定 2-4 个角色（太多会太吵）
3. **活跃度搭配**：至少留 1 个低活跃角色平衡气氛

### ❌ 错误做法
1. ❌ 不标记真人马甲 → AI 会生成你的发言
2. ❌ 全部设高活跃 → 论坛会非常吵闹
3. ❌ 只绑定 1 个角色 → 缺少多样性

##  常见问题

### Q1: 为什么勾选不了？
**A:** 已修复！使用原生 checkbox，现在可以正常勾选了 ✓

### Q2: 活跃度选择器不显示？
**A:** 只有勾选角色后才会显示，请确保已勾选复选框

### Q3: 设置了低活跃为什么角色还是经常出现？
**A:** AI 生成有一定随机性，但会明显减少低活跃角色的发言频率

### Q4: 数据会保存吗？
**A:** 会自动保存到本地存储，刷新页面/关闭浏览器都保留

### Q5: 可以绑定多少个角色？
**A:** 理论上无限制，但建议 2-4 个（太多会影响体验）

## 📊 数据持久化

### 存储位置
```javascript
// 绑定角色 ID 列表
forumStore.boundCharIds = ['chat_1', 'chat_2', ...]

// 角色活跃度
chatStore.chats['chat_1'].forumActivity = 'high'
```

### 保存时机
- ✅ 勾选/取消勾选时立即保存
- ✅ 更改活跃度时立即保存
- ✅ 关闭页面时自动保存
