#  桥桥手机 (QiaoQiao Phone)

一个基于 Vue 3 + Vite + Pinia 构建的**虚拟手机操作系统模拟器**，提供高度仿真的智能手机界面和完整的生态系统应用。

**当前版本**: v1.3.67  
**技术栈**: Vue 3.5.24 + Vite 5.4.11 + Pinia 3.0.4 + TailwindCSS 3.4.0

---

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🎯 核心功能

### 9 大核心应用
1. **微信** - 完整社交系统 (聊天/朋友圈/红包/通话)
2. **购物** - 电商系统 (商品/订单/物流/客服) ✨ 最新优化
3. **钱包** - 金融系统 (支付/账单/家庭卡)
4. **游戏** - 麻将对战 (AI 对战/在线匹配)
5. **日历** - 多功能工具 (日程/倒计时/日记)
6. **手机检测** - 系统工具 (设备信息/性能测试)
7. **设置** - 系统配置 (API/个性化/数据管理)
8. **微博** - 社交资讯 (热点新闻)
9. **世界观** - 剧情系统 (角色管理/轮回系统)

## 🛠️ 技术栈

- **框架**: Vue 3.5.24 (Composition API + `<script setup>`)
- **构建**: Vite 5.4.11 (快速 HMR + PWA)
- **状态管理**: Pinia 3.0.4 (模块化设计)
- **路由**: Vue Router 4.6.4 (动态路由)
- **样式**: TailwindCSS 3.4.0 (原子化 CSS)
- **图标**: FontAwesome 7
- **AI**: OpenAI / MiniMax / Pollinations
