### 紧急修复：HTML/CSS代码泄露问题（用户截图反馈）

**问题现象**（用户提供截图）：

- `getUnifiedCleanContent()` 只剥离了 HTML 标签，但标签内的属性文本和 CSS 代码块残留

**修复文件**：

| 文件 | 修改 |
|------|------|
| **SystemLogs.vue** | `formatDetail()` + `formatPayloadContent()` 添加 HTML 转义 + `\n` 换行 + CSS/代码块清洗 |
| **ChatMessageItem.vue** | `formatMessageContent()` 增加后处理：移除 style 属性、CSS 键值对、@keyframes、HTML 标签残留 |
| **FavoriteDetailView.vue** | `cleanMessage()` 从几乎无操作 → 完整 CSS/HTML 清洗；`renderMarkdown()` 添加预处理 |
| **chatMessageDisplay.js** | `getUnifiedCleanContent()` AI 清洗路径增加：style 属性剥离、40+ 种 CSS 属性键值对清理、短代码块检测清除、@keyframes 移除 |

## 任务7：紧急修复 — HTML代码/CSS残留泄露问题（聊天+日志）

### 问题现象
- **系统日志界面**：`style=`, `<div>`, `<span>` 等 HTML 代码和 `\n` 字面量直接显示

  - 清理 CSS 代码块 `{ ... }`

  - 清理多余空格/空白行

#### 2. ChatMessageItem.vue — 聊天消息渲染
- `formatMessageContent()` 在 `getCleanContent()` 之后增加防御清洗
- 覆盖 CSS 属性、HTML 标签属性、裸露标签、空白行清理

#### 3. FavoriteDetailView.vue — 收藏详情页
- `cleanMessage()` 从几乎无操作 → 完整的 CSS/HTML 清洗逻辑
- `renderMarkdown()` 增加 marked.parse 前的预处理步骤

#### 4. SystemLogs.vue — 系统日志（顺手修复）
- `formatDetail()` 和 `formatPayloadContent()` 添加 HTML 转义 + `\n` 转换 + CSS 清洗
