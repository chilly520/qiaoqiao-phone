# 火山引擎语音合成 (TTS) 配置教程

## 1. 注册与登录
访问 [火山引擎控制台](https://console.volcengine.com/) 并注册/登录您的账号。

## 2. 开通语音技术服务
1. 在控制台首页搜索或找到 **"语音技术" (Speech & Audio)** 服务。
2. 点击 **"立即开通"** (如果未开通)。
3. 在左侧菜单中选择 **"语音合成" (Speech Synthesis)**。

## 3. 创建应用 (获取 AppID)
1. 进入 **"应用管理" (Application Management)**。
2. 点击 **"创建应用"**。
3. 填写应用名称 (例如: `ChillyPhone`) 和描述。
4. 创建成功后，您将在列表中看到该应用。
5. 复制列表中的 **AppID** (例如: `1234567890`)，这是您需要填入手机 App 中的第一个参数。

## 4. 获取访问令牌 (AccessToken)
为了安全起见，我们推荐使用 **Token 认证** 方式调用接口。

1. 在应用列表中，点击您刚创建的应用名称进入详情页，或者查找 **"授权管理"** / **"Token 管理"** 选项。
2. 找到 **AccessToken (访问令牌)** 部分。
3. 如果没有现成的 Token，点击 **"新建 Token"** 或 **"生成 Token"**。
4. 复制生成的 **Token** (通常是一个长字符串)，这是您需要填入手机 App 中的第二个参数。

## 5. 在《小手机》中配置
1. 打开《小手机》App，进入 **设置** -> **语音设置**。
2. 在 **"语音引擎"** 列表中选择 **"火山付费版 (Pro)"**。
3. 在下方的配置区域中：
    - **AppID**: 填入第 3 步获取的 AppID。
    - **AccessToken**: 填入第 4 步获取的 Token。
4. 点击 **"保存设置"**。

## 6. 测试发音
1. 点击上方的 **"测试发音"** 按钮。
2. 如果配置正确，您将听到测试语音。
3. 如果听到错误提示，请检查 AppID 和 Token 是否填写正确，或确保账户余额充足。

## 常见问题
- **Q: 为什么提示鉴权失败 (Auth Failed)?**
  A: 请检查 AppID 和 Token 是否匹配，或者 Token 是否已过期 (部分 Token 有有效期)。
- **Q: 为什么没有声音？**
  A: 检查手机是否静音，或尝试切换网络。
- **Q: 哪里可以看剩余额度？**
  A: 在火山引擎控制台的 "费用中心" 或 "资源包管理" 中查看。
