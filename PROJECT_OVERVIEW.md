# 桥桥手机 (QiaoQiao Phone) - 项目介绍文档

## 📱 项目概述

**桥桥手机**是一个基于 Vue 3 + Vite + Pinia 构建的**虚拟手机操作系统模拟器**，提供高度仿真的智能手机界面和完整的生态系统应用。项目采用现代化前端技术栈，实现了包括社交、购物、金融、娱乐、工具等在内的完整应用矩阵。

**当前版本**: v1.3.67  
**技术栈**: Vue 3.5.24 + Vite 5.4.11 + Pinia 3.0.4 + Vue Router 4.6.4  
**UI 框架**: TailwindCSS 3.4.0 + FontAwesome 7.1  
**PWA 支持**: vite-plugin-pwa 0.20.5

---

## 🏗️ 项目架构

### 核心技术栈
- **前端框架**: Vue 3 (Composition API + `<script setup>`)
- **状态管理**: Pinia (多 Store 模块化设计)
- **路由系统**: Vue Router 4 (动态路由 + 错误处理)
- **构建工具**: Vite 5 (快速 HMR + PWA 打包)
- **样式方案**: TailwindCSS 3 + 自定义主题系统
- **图标系统**: FontAwesome 7

### 目录结构
```
src/
├── assets/              # 静态资源 (主题、动画、样式)
├── components/          # 全局组件 (通话、头像、音乐播放器等)
├── composables/         # 组合式函数 (聊天事务处理)
├── router/              # 路由配置 (25+ 路由)
├── stores/              # Pinia 状态管理 (15+ 模块)
│   ├── chatModules/     # 聊天 AI 模块 (群聊、财务、历史等)
│   └── ...              # 各应用状态管理
├── utils/               # 工具函数 (AI 服务、天气、日志等)
│   ├── ai/              # AI 相关 (提示词、请求队列)
│   ├── mahjong/         # 麻将游戏逻辑
│   └── ...
├── views/               # 页面视图 (按应用分类)
│   ├── Calendar/        # 日历应用
│   ├── Games/           # 游戏中心 (麻将对战)
│   ├── PhoneInspection/ # 手机检测系统
│   ├── Settings/        # 设置中心
│   ├── Shopping/        # 电商平台
│   ├── Wallet/          # 钱包金融
│   ├── WeChat/          # 社交应用
│   └── ...
├── App.vue              # 根组件 (状态栏、全局弹窗)
└── main.js              # 入口文件 (错误监控、通知服务)
```

## 📦 核心功能模块

### 1. **社交系统 (WeChat)** 📱
**文件**: `src/views/WeChat/`, `src/stores/chatStore.js`

- **即时通讯**: 支持文字、表情、图片、语音消息
- **AI 聊天**: 集成多种 AI 角色 (客服、群聊、财务助手等)
- **朋友圈**: 动态发布、点赞、评论、分享
- **群组功能**: 群公告、投票、任务调度
- **红包/转账**: 家庭卡、礼品卡、资金往来
- **收藏系统**: 消息收藏、整理、分类
- **角色系统**: 每个聊天对象有独立人设和记忆
- **通话功能**: 语音/视频通话界面

**特色功能**:
- AI 主动聊天 (Proactive Chat)
- 聊天记录永久保存
- 群聊 AI 机器人集成
- 财务聊天模块 (自动记账)

---

### 2. **电商系统 (Shopping)** 🛒
**文件**: `src/views/Shopping/`, `src/stores/shoppingStore.js`

- **用户中心**: 地址管理、收藏、足迹、优惠券、积分

## 🛠️ 开发指南

### 环境搭建
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

### 代码规范
- 使用 Vue 3 Composition API
- 遵循 `<script setup>` 语法
- 状态管理使用 Pinia
- 样式使用 TailwindCSS 原子类
- 组件命名采用 PascalCase

### 添加新应用
1. 在 `src/views/` 创建应用目录
2. 在 `src/stores/` 创建状态管理
3. 在 `src/router/index.js` 添加路由
4. 在 `HomeView.vue` 添加桌面图标
